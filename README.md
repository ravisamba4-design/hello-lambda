# Hello Lambda ☁️

A simple AWS Lambda function demonstrating serverless computing — no server management required. Exposed publicly via a Lambda Function URL, acting as a lightweight REST API.

## What it does
Returns a "Hello World" style response when invoked — accessible via a public HTTPS endpoint.

https://altgg3ufbxyow2v7shbwsfwdpa0icfps.lambda-url.eu-north-1.on.aws/

## What I learned
- Creating and deploying a Lambda function via AWS Console
- Understanding serverless architecture — no server provisioning needed
- Exposing a Lambda function publicly using Function URLs
- Basics of how API Gateway/Lambda patterns work in production systems
