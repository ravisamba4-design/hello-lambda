# Hello Lambda ☁️

A simple AWS Lambda function demonstrating serverless computing — no server management required.

## What it does
Returns a "Hello World" style response when invoked, showing basic Lambda function structure and event handling.

## What I learned
- Creating and deploying a Lambda function via AWS Console
- Understanding serverless architecture — no server provisioning needed
- Testing Lambda functions with custom test events
- Synchronous vs asynchronous invocation types
